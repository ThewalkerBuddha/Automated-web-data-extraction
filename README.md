# Automated web data extraction

Automated web data extraction as self explanatory naming is to automate extraction of data from webpages. there are primary two codes
1. PubMed crawler which is based on Bio Entrez package
2. Selenium based web crawler where it uses selenium browser with more humanoid interaction with web pages
 selenium based web extraction look longer intense development time to work beyond approximately 290 trackers enabled by Google. This was developed way before ChatGPT or any other LLMs were even a thing
